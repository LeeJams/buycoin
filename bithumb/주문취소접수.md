| 항목     | 내용           |
| -------- | -------------- |
| 이름     | 주문 취소 접수 |
| Method   | DELETE         |
| 요청 URL | /v1/order      |

| 구분           | 필드명 | 설명               | 타입   |
| -------------- | ------ | ------------------ | ------ |
| Request(Query) | uuid   | 취소할 주문의 UUID | String |

| 구분           | 필드명           | 설명                     | 타입         |
| -------------- | ---------------- | ------------------------ | ------------ |
| Response(Body) | uuid             | 주문의 고유 아이디       | String       |
| Response(Body) | side             | 주문 종류                | String       |
| Response(Body) | ord_type         | 주문 방식                | String       |
| Response(Body) | price            | 주문 당시 화폐 가격      | NumberString |
| Response(Body) | state            | 주문 상태                | String       |
| Response(Body) | market           | 마켓의 유일키            | String       |
| Response(Body) | created_at       | 주문 생성 시간           | DateString   |
| Response(Body) | volume           | 사용자가 입력한 주문 양  | NumberString |
| Response(Body) | remaining_volume | 체결 후 남은 주문 양     | NumberString |
| Response(Body) | reserved_fee     | 수수료로 예약된 비용     | NumberString |
| Response(Body) | remaining_fee    | 남은 수수료              | NumberString |
| Response(Body) | paid_fee         | 사용된 수수료            | NumberString |
| Response(Body) | locked           | 거래에 사용중인 비용     | NumberString |
| Response(Body) | executed_volume  | 체결된 양                | NumberString |
| Response(Body) | trades_count     | 해당 주문에 걸린 체결 수 | Integer      |
