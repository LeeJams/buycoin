| 항목     | 내용              |
| -------- | ----------------- |
| 이름     | 주문 가능 정보    |
| Method   | GET               |
| 요청 URL | /v1/orders/chance |

| 구분           | 필드명 | 설명                  | 타입   |
| -------------- | ------ | --------------------- | ------ |
| Request(Query) | market | 마켓 ID (예: KRW-BTC) | String |

| 구분             | 필드명        | 설명                  | 타입         |
| ---------------- | ------------- | --------------------- | ------------ |
| Response(최상위) | bid_fee       | 매수 수수료 비율      | NumberString |
| Response(최상위) | ask_fee       | 매도 수수료 비율      | NumberString |
| Response(최상위) | maker_bid_fee | 마켓 매수 수수료 비율 | NumberString |
| Response(최상위) | maker_ask_fee | 마켓 매도 수수료 비율 | NumberString |
| Response(최상위) | market        | 마켓 정보             | Object       |
| Response(최상위) | bid_account   | 매수 화폐 계좌 상태   | Object       |
| Response(최상위) | ask_account   | 매도 화폐 계좌 상태   | Object       |

| 구분   | 필드명      | 설명                                               | 타입          |
| ------ | ----------- | -------------------------------------------------- | ------------- |
| market | id          | 마켓 유일 키                                       | String        |
| market | name        | 마켓 이름                                          | String        |
| market | order_types | 지원 주문 방식                                     | Array[String] |
| market | ask_types   | 매도 주문 지원 방식                                | Array[String] |
| market | bid_types   | 매수 주문 지원 방식                                | Array[String] |
| market | order_sides | 지원 주문 종류                                     | Array[String] |
| market | bid         | 매수 제약사항 (currency, price_unit, min_total 등) | Object        |
| market | ask         | 매도 제약사항 (currency, price_unit, min_total 등) | Object        |
| market | max_total   | 최대 매도/매수 금액                                | NumberString  |
| market | state       | 마켓 운영 상태                                     | String        |

| 구분        | 필드명                 | 설명                   | 타입         |
| ----------- | ---------------------- | ---------------------- | ------------ |
| bid_account | currency               | 화폐 코드              | String       |
| bid_account | balance                | 주문가능 금액/수량     | NumberString |
| bid_account | locked                 | 주문 중 묶인 금액/수량 | NumberString |
| bid_account | avg_buy_price          | 매수평균가             | NumberString |
| bid_account | avg_buy_price_modified | 매수평균가 수정 여부   | Boolean      |
| bid_account | unit_currency          | 평단가 기준 화폐       | String       |

| 구분        | 필드명                 | 설명                   | 타입         |
| ----------- | ---------------------- | ---------------------- | ------------ |
| ask_account | currency               | 화폐 코드              | String       |
| ask_account | balance                | 주문가능 금액/수량     | NumberString |
| ask_account | locked                 | 주문 중 묶인 금액/수량 | NumberString |
| ask_account | avg_buy_price          | 매수평균가             | NumberString |
| ask_account | avg_buy_price_modified | 매수평균가 수정 여부   | Boolean      |
| ask_account | unit_currency          | 평단가 기준 화폐       | String       |
