## 기본 정보

| 항목     | 내용                                |
| -------- | ----------------------------------- |
| 이름     | 전체 계좌 조회                      |
| Method   | GET                                 |
| 요청 URL | https://api.bithumb.com/v1/accounts |

## Request 값

- Query String: 없음.
- Body: 없음 (GET 호출).

## Response 값 (200 성공 시 각 계좌 항목)

| 필드명                 | 설명                             | 타입         |
| ---------------------- | -------------------------------- | ------------ |
| currency               | 화폐를 의미하는 영문 대문자 코드 | String       |
| balance                | 주문가능 금액/수량               | NumberString |
| locked                 | 주문 중 묶여있는 금액/수량       | NumberString |
| avg_buy_price          | 매수평균가                       | NumberString |
| avg_buy_price_modified | 매수평균가 수정 여부             | Boolean      |
| unit_currency          | 평단가 기준 화폐                 | String       |

에러 시에는 400 등 HTTP Status와 함께 에러 응답 스키마(Result)가 반환됩니다.
